# Biofilia Web (Next.js)

## 1) Requisitos
- Node.js 18+

## 2) Instalación
```bash
npm install
```

## 3) Desarrollo
```bash
npm run dev
```

Abrí: http://localhost:3000

## 4) Dónde poner imágenes
- `public/images/...`
- Referencialas desde `src/data/slides.*.ts`

## 5) Ajustes rápidos
- Tu número de WhatsApp: `src/lib/site.ts` (phoneE164)
